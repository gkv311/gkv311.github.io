sView 25.02 избавилась от поддержки некоторых устаревших компонентов и платформ:
- Удалены *32-битные* бинарные сборки из Windows (теперь доступны только *x64* сборки);
- Удалены библиотеки *FreeImage* и *DevIL* из пакета для Windows,
  а также улучшено чтение **HDR**/**EXR** изображений средствами *FFmpeg*;
- Удалён NPAPI/ActiveX плагин (браузеры прекратили поддержку таких плагинов 2015-2018);
- DMG пакет для macOS теперь содержит *ARM64* бинарные сборки вместо *x64*.

Добавлена поддержка нового *FFmpeg 7.1* API для декодирования видео с чередующимися стереоскопическими видепотоками.

Кроме того, среда разработки sView была переведена на *CMake* скрипты.
