Для Linux дистрибутивов, основанных на Debian

Официальный репозиторий sView для Ubuntu расположен на Launchpad. После добавления PPA в систему, установить sView можно следующей командой в консоли:

sudo apt-get install sview


Для запуска sView из командной строки воспользуйтесь одним из следующих вариантов:

sView --in=image

sView --in=video

sView media_file_name

Дополнительная информация по установке на OS X

Бинарные файлы sView поставляются без цифровой подписи. Отсутствие цифровой подписи несколько затрудняет установку программы на новых версиях системы в следствие высокого уровня безопасности, установленного в системе по-умолчанию (блокирующего запуск неподписанных приложений). Для запуска sView необязательно менять настройки безопасности системы! Вы можете просто добавить приложение в исключения:

  • Установите sView как и любое другое приложение (перетащите "sView.app" в "/Applications").
  • Перейдите в каталог "/Application" используя файловый менеджер Finder.
  • Выберете пункт "Open" в контекстном меню файлового менеджера для файла "sView.app", удерживая клавишу Shift.
  • Кликните "Open" в появившемся окне для игнорирования предупреждения для неподписанного приложения (в случае запуска приложения обычным способом без контекстного меню или не удерживая клавишу Shift, кнопки "Open" не будет!).

Эту процедуру необходимо выполнить только один раз сразу после установки. После этого вы можете использовать sView без каких либо ограничений!
