Данная страница содержит только существенные изменения, заметные конечному пользователю. Если вы ищете полную историю - пожалуйста просмотрите историю интеграций в репозитарии git.

sView 20.08

20.08 (2020-08-01)

+ Добавлена поддержка равноугольных кубических карт (EAC).

* Исправлены различные ошибки приводящие к искажению изображения на HTC Vive.

* Android, улучшена доступность некоторых функций (пролистывание длинных меню в альбомной ориентации экрана, настройки размера и позиции субтитров).

* macOS, для декодирования видео аппаратным декодером теперь используется VideoToolbox вместо VDA.

* macOS, для запуска теперь требуется macOS версии 10.10 Yosemite и выше (предыдущей релиз sView 17.04 поддерживал OS X 10.6+ Snow Leopard).

20.05 (2020-05-20)

+ Добавлены файлы испанского перевода.

+ Image Viewer - добавлена опция трилинейной фильтрации текстур.

+ Image Viewer - поддержка чтения кубических карт упакованных в порядке 1x6.

+ Image Viewer - автоматическая активация панорамного вывода для JPEG файлов с тегом GPano:ProjectionType.

+ Image Viewer - автоматическая активация панорамного вывода для JPEG файлов с тегом 360Stereo/360Mono (скриншоты видеоигр).

+ Image Viewer - чтение кубических карт упакованных в контейнер DDS.

+ Movie Player - возможность задать размер нетекстовых субтитров.

* Movie Player - улучшена перемотка SRT субтитров, загруженных из внешнего файла.

* Movie Player - исправлено форматирование субтитров в формате ASS с незакрытыми тегами.

* Movie Player - добавлена обработка нетекстовых субтитров записанных в стереоформате видеоряда (бок о бок).

* Image Viewer - улучшено качество сохранения JPEG (yuv420p) изображений в PNG (rgb) формат.

* Image Viewer - исправлено отображение WebP изображений с прозрачностью (yuva формат пикселя).

* Библиотека fontconfig теперь используется для поиска системных шрифтов на Linux.

19.08 (2019-08-04)

+ Добавлена поддержка VR180 (полусфера) и цилиндрических панорам.

+ Image Viewer теперь автоматически открывает видеофайлы в Movie Player и наоборот.

+ Добавлен жест для перехода к следующему/предыдущему файлу.

+ Совместимость с FFmpeg 4.1.

+ Android, добавлена поддержка воспроизведения музыки при работе sView в фоне.

* Android, добавлена пометка поддержки ултра-широких экранов.

* Android, доступ к SD карте теперь запрашивается при старте приложения для соответствия новым требованиям Google Play.

* Android, исправлено падение приложения при запуске на некоторых устройствах Huawei (NULL-dereference в системном компоненте), получивших обновление до Android 9.

* Android, улучшена производительность воспроизведения 4K видео на устройствах с графикой Adreno (загрузка текстуры).

* Android, поддержка скролла мыши.

* Windows, исправлен скроллинг на тачпадах поддерживающих плавный скролл.

* Совместимость с Ubuntu 19.04.

* Исправлена редкие спорадические ошибки форматирования текста.

17.10 (2017-10-23)

+ Image Viewer - добавлена опция настройки интервала слайдшоу.

+ Movie Player - добавлена информация о потоке (битрейт, соотношение сторон) к информации о файле.

+ Android, добавлена поддержка автостереоскопического планшета Commander 3D.

+ Android, добавлено детектирование PPTV King 7S со стереоэкраном.

+ Android, Image Viewer - добавлена навигация по списку воспроизведения посредством кнопок регулирования громкости.

* Movie Player - исправлена проблема перемотки файла к самому началу.

* Movie Player - исправлена проблема отображения изображения, прикреплённого к некоторым mp3 файлам.

* Исправлено некорректное применение кернинга при отображении текста.

* Исправлена ошибка компиляции при использовании glibc 2.26+ (Ubuntu 17.10+).

* Android, вторая (внешняя) SD карта теперь отображается в диалоге открытия файла.

17.04 (2017-04-16)

+ Movie Player теперь автоматически поворачивает видео при наличии такой информации в файле (портретная съёмка смартфоном).

+ Image Viewer, Movie Player - добавлен контрол для цветокоррекции.

* Image Viewer, Movie Player - исправлена проблема зависания (слишком длительной обработки) диалога информации о файле при наличии длинного текста в метаданных.

17.01 (2017-01-10)

+ Добавлена поддержка библиотеки OpenVR, включая позиционирование головы при выводе сферических панорам (тестировалось на HTC Vive).

+ Movie Player - добавлено позиционирование звука при просмотре сферических панорам.

* Исправлена проблема сохранения PNG изображений на актуальной версии FFmpeg.

* Улучшена поддержка смешанных HiDPI конфигураций мониторов на Windows 10 (добавлена использование нового API, требующего летнего ('2016) обновления Windows 10).

* Установочный образ для OS X теперь поставляется с OpenAL Soft (ранее использовалась OpenAL в составе системы - от Apple).

* Movie Player - добавленая опция форсированного включения HRTF реализованного в OpenAL Soft.

* Удалён код поддержки системы Windows XP.

16.06 (2016-07-02)

+ FFmpeg обновлён до версии 3.1.1.

+ Android, добавлена поддержка множественных касаний.

+ Android, улучшена совместимость с файловыми менеджерами и почтовыми клиентами.

+ Android, добавлено частичное аппаратно-ускоренное декодирование видео (отключено по-умолчанию).

+ Movie Player - добавлена конверсия цвета на GPU для форматов NV12 и XYZ12.

+ Image Viewer - диалог сохранения файла теперь предлагает название оригинального файла по-умолчанию.

* Movie Player - исправлена ошибка сохранения позиции воспроизведения в списке недавно открытых файлов.

* Movie Player - исправлена ошибка обработки статуса воспроизведения видео для сохранения экрана включенным.

* Android, исправлено аварийное завершение программы при закрытии на некоторых устройствах.

* Улучшено переключение языка интерфейса без перезапуска программы.

* Программа теперь предотвращает открытие нескольких одинаковых или эксклюзивных диалогов.

15.11 (2015-11-22)

+ Movie Player - добавлена возможность удалённого управления через аргументы программы.

+ Image Viewer - поддержка sTER чанка в PNG файлах идентифицирующего параллельную стереопару.

+ Image Viewer - добавлено отображение метаданных, прочитанных FFmpeg.

+ Добавлена поддержка списка файлов для Drag & Drop. Список из двух файлов открывается как стереопара.

+ Android, добавлена поддержка горячих клавиш.

+ OpenAL Soft обновлён до версии 1.17.

+ FFmpeg обновлён до версии 2.8.2.

+ Код поддержки Oculus Rift обновлён до SDK 0.8.

* Уменьшено пиковое потребление памяти, улучшена производительность за счёт использования изображений без копирования памяти.

* Windows, обновлена экспериментальная поддержка DXVA2 декодера (поддерживаются больше кодеков).

* Windows, исправлено скрытие курсора в NVIDIA 3D Vision выводе (Direct3D).

* Android, исправлена проблема совместимости с некоторыми устройствами OpenGL ES 2.0, поддерживающими расширение GL_EXT_texture_rg (не отображался интерфейс программы).

* Клавиша Esc теперь закрывает любое контекстное меню.

15.10 (2015-10-04)

+ Добавлена поддержка кубмап в формате 3:2 (две строки по 3 стороны куба).

+ На панель горячих действий добавлена кнопка для активации просмотра панорамы (кубических и сферических на 360 градусов).

+ Добавлена частичная чешская локализация.

+ Android, добавлена поддержка управлением панорамного просмотра с помощью сенсоров ориентации устройства (требуются точные сенсоры!).

+ Android, реализована обработка кнопки "Назад" устройства для закрытия текущего диалога.

+ Добавлена опция для автоматической коррекции соотношения сторон анаморфного 1080p и 720p видео (включена по-умолчанию).

* Windows, функция блокировки выключения монитора теперь отключается при явной блокировке сессии пользователя.

* Файл, выбранный из автоматически сгенерированного списка воспроизведения, теперь добавляется в список последних файлов перед закрытием программы.

* Android, исправлены проблемы сохранения настроек и восстановления последнего открытого файла при различных сценариях.

* Улучшение обработки файлов настроек на Linux и Android - убраны лишние операции чтения / записи файла.

15.08 (2015-08-29)

+ Поддержка стереоскопических кубмап (панорамное изображение заданное в виде 6ти граней куба).

+ Добавлен диалог для задания горячих клавиш.

+ Добавлена панель управления громкостью.

+ Добавлены кнопки переключения последовательности проигрывания элементов в списке видеопроигрывателя.

* Исправлено масштабирование интерфейса в многомониторных конфигурациях с HiDPI монитором на Windows 8.1 и Windows 10.

* Обновлённый интерфейс - более компактный, новые иконки в меню, полоса прокрутки, улучшенная поддержка HiDPI мониторов.

* Исправлены проблемы интеграции в Ubuntu (отсутствие иконки на панели задач).

* Улучшенная поддержка пользовательского ввода пальцами.

* Множество исправлений ошибок.

14.11 (2014-11-30)

+ Портирование на FFmpeg 2.5.

+ Windows, вывод 3D Vision через Direct3D - добавлена поддержка расширения WGL_NV_DX_interop для улучшения производительности; улучшена стабильность.

+ Предварительная поддержка платформы Android.

+ Добавлены файлы перевода для китайского языка.

+ Movie Player - по-умолчанию проигрыватель теперь выбирает звуковой поток с учётом выбранного языка для интерфейса.

+ Movie Player - добавлена поддержка аудиопотоков 3.0.

* Windows - исправлено искажение размеров окна при смене конфигурации мониторов.

* OS X - заголовок окна теперь восстанавливается при выходе из полноэкранного режима.

* Movie Player - улучшено воспроизведение в случайном порядке.

* Movie Player - улучшена поддержка m3u списков воспроизведения.

* Movie Player - исправлена перемотка mp3 файлов с прикреплённым изображением.

* Movie Player - добавлена обработка не поддерживаемых и неправильных форматов звуковых дорожек.

* Image Viewer - приложение теперь масштабирует большие изображения, размеры которых превышают лимиты текстур используемой видеокарты.

* Image Viewer - исправлена ошибка чтения некоторых JPEG файлов.

14.02 (2014-03-01)

+ Movie Player - реализовано возобновление проигрывания продолжительных видеофайлов (> 12 минут) с позиции, при которой файл был закрыт.

+ Movie Player, Image Viewer - реализовано автоопределение стереоформата по имени файла по ключевым словам abq/lrq/halfou/halfsbs (используется только в том случае, когда файл не содержит соответствующей информации).

- Movie Player, Image Viewer - отключено автоопределение стереоформата перекрёстной пары по соотношению сторон.

14.01 (2014-01-30)

+ Windows 8.1 - размеры окна теперь масштабируются при переносе на монитор с другим DPI.

+ Windows, чересстрочный вывод - позиция окна выравнивается к чётным числам, чтобы при переносе окна ракурсы не сбивались (до перерисовки содержимого).

+ Movie Player - при добавлении звуковой дорожки / субтитров из дополнительного файла, она автоматически выбирается.

+ Movie Player - добавлена поддержка не текстовых субтитров (картинок).

+ Movie Player - добавлена поддержка базовых тегов форматирования HTML (< b > и < i >) в текстовый субтитрах.

+ Movie Player - текст отображаемых на экране субтитров теперь может быть скопирован в буфер обмена комбинацией Ctrl + C.

+ Movie Player - добавлен механизм автоматического добавления дополнительных звуковых дорожек и субтитров из файлов с похожим названием.

+ Image Viewer - добавлены чтение и запись стереопометок в JPEG файлах _JPSJPS_.

+ Image Viewer - добавлено уведомление о проблема с метаданными в файлах (отсутствие таковых или не соответствие выбранному вручную формату) в диалог информации о файле.

+ Добавлена поддержка нового stereo3d API в FFmpeg.

+ Добавлен частичный немецкий перевод.

+ Добавлен частичный корейский перевод.

+ В добавок к западным шрифтам теперь также загружаются корейские и CJK при необходимости.

* Улучшено отображение встроенных диалогов. Расширена информация в диалоге "О файле".

* Image Viewer - пропорции пикселя, сохранённые в файле, теперь применяются и для изображений.

* Image Viewer - улучшена поддержка MPO (из файлов, содержащих большие эскизы, теперь читается пара изображений с максимальным размером).

* Image Viewer - исправлены ошибки в обработке содержимого JPEG файлов.

* Movie Player - исправлено воспроизведение 5.0 и 7.1 звуковых дорожек.

* Oculus Rift теперь представлен как отдельное устройство вывода для упрощения настройки.

* Исправлена ошибка выравнивания букв при повторном выводе того же текста.

* OS X 10.9 - исправлен крах, возникающий при вызове системного фреймворка OpenAL с пустым буфером.

* Linux - улучшен алгоритм поиска шрифтов.

* Linux - исправлено открытие последних файлов из меню Gnome (добавлена поддержка протокола file:// в параметрах коммандной строки).

13.10 (2013-10-05)

+ Windows - добавлена поддержка DPI-awareness для Windows Vista+ и поддержка нового API с независимым DPI для разных мониторов введённая в Windows 8.1.

+ Linux, XLib - добавлено чтение Xft.dpi для автоматической подстройки масштабирования интерфейса.

+ Distorted Output, Dual Output - добавлена опция для вывода моно источников в режиме стерео.

+ Movie Player - добавлены опции для выбора размера текста и параллакса субтитров.

* Movie Player - исправлена регрессия (кнопка 'W' не меняла местами ракурсы).

^ Оптимизирована загрузка текста.

13.08 (2013-08-18)

+ Mac OS X - добавлена поддержка мониторов с высокой плотностью пикселей (Retina).

+ Добавлена опция масштабирования интерфейса.

+ Видеоплеер - добавлено функция синхронизации видео со звуком.

+ Видеоплеер, Web UI - добавлено управление громкостью звука.

+ Добавлены иконки для меню "исходный формат".

* Исправлена ошибка при удалении последнего файла в списке воспроизведения.

* Mac OS X - исправлено зависание программы при переходе из полноэкранного режима в оконной на системе Mountain Lion.

* Mac OS X - исправлено некорректная обработка удержания клавиши.

* Разнообразные исправления в интерфейсе.

* Исправлена версия в DLL файлах.

13.07 (2013-07-28)

+ Портирование на FFmpeg 2.0 (новые API, планарные аудиоформаты).

+ Извлечение информации об исходном стереоформате для каждого видеокадра, когда поддерживается кодеком.

+ Mac OS X - добавлена экспериментальная поддержка декодирования с помощью VDA (с задействованием GPU).

* Ускорено преобразование расширенных планарных форматов YUV (9/10/16 бит на компоненту) в RGB (используя GLSL).

* Файлы с двумя видеопотоками теперь выводятся как стереопара только при выбранном исходном формате "Автоопределение".

* Обход проблем с wglMakeCurrent() в некоторых OpenGL драйверах на Windows (в некоторых случаях возникали утечки памяти).

13.06 (2013-06-15)

+ Видеоплеер - реализован удалённый контроль через встроенный HTTP-сервер.

+ Добавлена предварительная поддержка Oculus Rift.

* Исправление ошибок.

13.05 (2013-05-25)

+ Видеоплеер - добавлено меню с недавно открытыми файлами.

+ Видеоплеер - добавлены горячие клавиши для переключения субтитров ('T') и звуковых дорожек ('L').

+ Видеоплеер - добавлен виджет для отображения плейлиста (Ctrl + L).

+ Видеоплеер - добавлено распознавание распространённых стереотегов в WMV файлов.

+ Добавлен Distorted модуль для анаморфного вывода.

* Linux - улучшена совместимость с XRandr для детектирования мониторов.

* Добавлен виджет для отображения FPS.

* Dual output - теперь полноэкранный режим пытается использовать одно окно для двух мониторов (для улучшения синхронизации).

* Плагин для браузера - исправлен полноэкранный режим в многомониторных конфигурациях.

* Плагин для браузера теперь не загружается на странице до тех пор, пока он не виден.

* Переключение модуля вывода теперь не прерывает проигрывание текущего файла.

* Оптимизировано использование видеопамяти (не создаётся буфер глубины, когда он не нужен).

* Улучшена обработка клавиатурного ввода.

* Улучшен виджета диалогового окна.

* m3u файлы более не добавляются в плейлист, исправлены несколько ошибок в m3u парсере.

13.01 (2013-01-07)

+ Настройка разделения ракурсов теперь возможна кнопками '<' и '>'.

+ Добавлена обработка смены конфигурации дисплеев при работающей программе.

+ Видеоплеер - добавлена поддержка дополнительных мультимедийных кнопок.

+ Видеоплеер - добавлена опция предотвращения автоматического отключения дисплея в следствие отсутствия активности.

+ Видеоплеер - обработка отключения аудио-устройства при работе программы, выбранного для вывода звука.

+ Видеоплеер - обрабатываются дополнительные теги в WMV видеофайлах для автоматического выбора исходного стереоформата.

+ Отображение грубой информации о текущем файле по кнопке 'I'.
