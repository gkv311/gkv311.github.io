sView for Debian-based Linux distributions

The official sView repository for Ubuntu is now placed on Launchpad. After adding PPA into your system you can install sView using this command from console:

sudo apt-get install sview


To launch sView from command line you should type one of the options:

sView --in=image

sView --in=video

sView media_file_name

Extra info for OS X users

sView has no digital signature and requires additional installation steps on OS X where unsigned applications are blocked by default. Please notice that you have not lower security level system-wide to launch sView! Instead you may add exception to concrete application:

  • Install sView as any application (drag "sView.app" to "/Applications").
  • Navigate to "/Application" in the Finder.
  • Open context menu for "sView.app" and choose "Open" (notice that you can not do this in Launchpad!).
  • Click "Open" in appeared dialog to ignore unsigned application warning (beware - if you launch application not from context menu then the same window will appear but without "Open" button!).

You need to perform this operation just once after installation. After this you may use sView without any restrictions!
